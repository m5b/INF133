A Brief History of JavaScript

  • Originally called LiveScript
  • Shipped with version 2.0 of Netscape Navigator
  • Later renamed to JavaScript to leverage increasing popularity of Java
  • Programming language for the web

A Brief History of JavaScript

Syntactically similar to Java, but:
  • Weakly typed (implicit typing based on operation)
  • Dynamically typed
  • Interpreted, not compiled
  • Though...can be transpiled or 'minified'
